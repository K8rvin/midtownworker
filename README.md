# Работяга из мидтауна

Браузерный симулятор городской жизни на Phaser 3: жильё, работа, голод и сон, курьерские доставки, магазины и сюжетные подсказки. Вдохновлено GTA (город сверху), The Sims (нужды, дом) и Монополией (аренда, накопления).

**План развития:** [docs/ROADMAP.md](docs/ROADMAP.md)

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5173

## Игра онлайн (GitHub Pages)

После push в `main` игра публикуется автоматически:

**https://k8rvin.github.io/midtownworker/**

Первый раз в настройках репозитория:

1. Откройте https://github.com/K8rvin/midtownworker/settings/pages
2. **Build and deployment → Source:** `Deploy from a branch`
3. **Branch:** `gh-pages` → папка `/ (root)` → **Save**
4. Подождите 1–2 минуты — появится зелёная ссылка на сайт

Для бесплатного аккаунта GitHub Pages работает только с **публичным** репозиторием (приватный — нужен GitHub Pro).

Локальная сборка как на Pages:

```bash
set GITHUB_PAGES=true
npm run build
npm run preview
```

## Звук и настройки

- **НАСТРОЙКИ** в главном меню или паузе
- Громкость эффектов и музыки (клик по полоске)
- Вкл/выкл звука, кнопка «Проверить звук»
- Настройки сохраняются в браузере (`localStorage`)

## Геймплей

- **3 жизни**, здоровье, деньги, wanted level (0–5)
- **3 банды**: Якудза, Реднеки, Учёные — шкала уважения
- **15 квестов** через таксофоны (доставка, убийства, угон, блокпосты и др.)
- **Блокпосты** на перекрёстках — копы задерживают при розыске
- **Враждебные банды** атакуют в своём районе при низком уважении
- **Машины**: седан, спорткар, грузовик, полицейская
- **Магазины**: оружие, автосалон, больница
- **2 уровня**: улица и крыши (лестницы)

## Мир и карта (фаза 3)

- Карта **Tiled** (`public/maps/city.tmj`) — слои ground, roof, zones
- **Spritesheet** `city_tileset` — все тайлы в одной текстуре
- **Навигация A\*** — трафик едет по дорогам, враждебные NPC преследуют по путям

```bash
npm run generate:map   # пересоздать city.tmj
```

## Визуал и атмосфера (фаза 9, в работе)

- Neo-noir ночной город: неон, виньетка, лёгкие scanlines
- Анимация ходьбы игрока (spritesheet 4 кадра)
- VFX: взрывы, вспышки выстрела, искры, дым от шин
- Улучшенные тайлы зданий и дорог

## Контент и карта (фаза 5)

- **Tiled object layer** — таксофоны, магазины, флаги и лестницы в `city.tmj` (слой `objects`)
- **Единый источник** — `src/data/world-objects.json` + `shops.json` для размещения на карте
- **Цепочка квестов** — 15 квестов открываются последовательно (`requiresQuest`)
- **Журнал квестов** — заблокированные квесты с 🔒 и подсказкой «Сначала: …»
- **Полная карта (M)** — иконки игрока, квеста, магазинов, таксофонов, блокпостов, копов

```bash
npm run generate:map   # пересоздать city.tmj с object layer
```

## Глубина и кампания (фаза 4)

- **Арест на блокпосту** — штраф $200, потеря жизни (как у копов)
- **Сохранение квестов** — эскорт, посылки, точки захвата и блокпосты восстанавливаются после F5 / «Продолжить»
- **Патроны** — докупка в оружейном для купленного оружия
- **Статус банд** — ✓ союзник / — нейтрал / ⚔ враг на HUD
- **Флаги районов** — на карте отмечены базы банд
- **Захват территории** — квест доступен только при уважении > −50
- **Полиция** — копы преследуют по A*; при розыске ≥3 появляются патрульные машины
- **Дробовик** — 5 дробинок за выстрел
- **Победа** — экран «Город покорён» после всех 15 квестов

## HUD (фаза 1)

- Слоты оружия 1–4 с патронами
- Шкалы уважения трёх банд
- HP машины при вождении
- Обучение при первом запуске
