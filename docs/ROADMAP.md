# План развития GTA 2 — Top Down

Статус на июль 2026. Фазы 1–15 (симулятор жизни) в работе.

## Завершённые фазы

| Фаза | Название | Содержание |
|------|----------|------------|
| **1** | HUD | Слоты оружия, шкалы банд, HP машины, обучение |
| **2** | Геймплей | Бой, 15 квестов, банды, полиция, машины, магазины, сохранения, звук |
| **3** | Мир и карта | Tiled `city.tmj`, A*, трафик по дорогам |
| **4** | Глубина и кампания | Аресты, сохранение квестов, патроны, полиция-машины, победа |
| **5** | Контент и карта | Object layer, цепочка квестов, полная карта с иконками |

## Фаза 6 — Полировка и UX *(завершена)*

**Цель:** удобство и «ощущение продукта» без смены движка.

- [x] Мобильное управление (виртуальный джойстик, кнопки E/огонь/спринт)
- [x] Гибридный звук (улучшенные процедурные SFX + victory fanfare)
- [x] Настройки: чувствительность руля, режим сенсорного управления, сброс клавиш
- [x] Контекстные подсказки при смерти / аресте / провале квеста
- [x] Game over / victory — время забега, рекорды
- [x] Миникарта — зоны банд, магазины, флаги

*Оценка: 1–2 недели*

## Фаза 7 — Контент и прогрессия *(завершена)*

- [x] Гараж — купленные машины сохраняются и спавнятся

*Оценка: 2–3 недели*
