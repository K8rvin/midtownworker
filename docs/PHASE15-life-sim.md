# Фаза 15 — Симулятор жизни

Статус: **в реализации** (июль 2026). Полная перестройка жанра: без насилия, банд и полиции.

## Цель

Игрок зарабатывает, платит за жильё, следит за **голодом** и **соном**, покупает еду, устраивается на работу, обустраивает **дом как отдельную локацию**.

## Ключевые решения (утверждено)

| Решение | Выбор |
|---------|--------|
| Мир | Hybrid: город 200×200 остаётся, фокус на 3–4 зонах (дом, работа, супермаркет, банк/риелтор) |
| Жильё | Сначала **аренда** (плата раз в 7 игровых дней), затем **покупка** квартиры |
| **Работа** | **Только одна работа одновременно** — нельзя совмещать должности |

---

## Что убираем

- Банды, уважение, территории, флаги банд
- Оружие, бой, wanted, полиция, блокпосты, аресты
- Враждебные NPC и насильственные квесты (kill, steal, escape, …)
- Крыши/угон — опционально; машины только как транспорт

## JobManager — одна работа

### Правила

1. **Максимум одна активная работа** в `state.job`.
2. Устройство на новую работу возможно только если `job.id === null` **или** после явного **увольнения** (меню / `[E]` у HR).
3. При устройстве показывать: «Вы уволитесь с текущей должности?» — если уже есть работа (на случай багов; в норме блокировать приём).
4. Удалёнка доступна только у должностей с `remote: true` и после условия (например, 3 отработанных дня).
5. Зарплата начисляется **раз в игровой день** после отработанной смены (офис) или действия «Работать дома» (remote).

### Данные `jobs.json` (черновик)

| id | Название | Локация | remote | $/день |
|----|----------|---------|--------|--------|
| courier | Курьер | Промзона | нет | 80 |
| developer | Программист | Центр | да (с 3-го дня) | 150 |
| cashier | Продавец | Супермаркет | нет | 60 |
| waiter | Официант | Набережная | нет | 70 |

### UI

- Без работы: `[E]` у работодателя → «Устроиться»
- С работой: тот же NPC → «Смена» / «Уволиться» (не «вторая работа»)
- Дом: кнопка «Работать удалённо» только если `job.remote &&` смена не требует офиса

## Остальные системы (кратко)

### NeedsManager
- Голод и сон падают со временем; еда / кровать восстанавливают
- При 0 — обморок (штраф деньгами), без смерти от насилия

### HousingManager
- Аренда: оплата каждые 7 дней; просрочка → выселение
- Покупка: накопление → владение, без арендной платы
- **HomeScene** — отдельная локация 8×6, дверь в город

### Магазины
- `grocery` — продукты (`groceries.json`)
- Мебель — `furniture.json`, слоты в доме

### Задачи жизни
- `life-tasks.json` вместо violent quests; доска объявлений вместо «квестов убийства»
